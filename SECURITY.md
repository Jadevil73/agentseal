# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in AgentSeal, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, email us at [hello@agentseal.org](mailto:hello@agentseal.org) with:

- A description of the vulnerability
- Steps to reproduce it
- The potential impact

We will acknowledge your report within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Supported Versions

| Version | Supported |
|---------|:---------:|
| 0.6.x (latest) | Yes |
| < 0.6.0 | No |

## Scope

This policy covers the AgentSeal open source packages:

- Python package (`agentseal` on PyPI)
- JavaScript package (`agentseal` on npm)
- This GitHub repository

For vulnerabilities in the AgentSeal dashboard or hosted services, email the same address.
